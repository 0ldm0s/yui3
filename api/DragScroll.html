Class DragScroll - extends Drag

This class extends the dd-drag module to add the ability to scroll the window.

Configuration Attributes

buffer - Number

The number of pixels from the edge of the screen to turn on scrolling. Default: 30

scrollDelay - Number

The number of milliseconds delay to pass to the auto scroller. Default: 90

scrollWindow - Boolean

Turn on window scroll support
