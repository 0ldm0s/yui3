Module: loader

Provides dynamic loading for the YUI library. It includes the dependency info for the library, and will automatically pull in dependencies for the modules requested. It supports rollup files (such as utilities.js, and will automatically use these when appropriate in order to minimize the number of http connections required to load all of the dependencies. It can load the files from the Yahoo! CDN, and it can use the combo service provided on that network.
