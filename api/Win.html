Class Win

A wrapper for DOM Windows. Window properties can be accessed via the set/get methods. With the exception of the noted properties, only strings, numbers, and booleans are passed through. Use Win.get() to create new Win instances.

Properties

document - Doc

Returns a Doc instance wrapping the window.document

frameElement - Doc

Returns a Node instance wrapping the window.frameElement

height - String

Returns the inner height of the viewport (exludes scrollbar).

width - String

Returns the inner width of the viewport (exludes scrollbar).

window - Doc

Returns a Win instance wrapping the window.window
