Class Node

A wrapper for DOM Nodes. Node properties can be accessed via the set/get methods. With the exception of the noted properties, only strings, numbers, and booleans are passed through. Use Y.get() or Y.Doc.get() to create Node instances.

Properties

A NodeList containing only HTMLElement child nodes
